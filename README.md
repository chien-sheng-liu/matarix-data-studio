## 🛠️ Tech Stack

### Frontend
- **Framework:** [Next.js 15](https://nextjs.org) with App Router
- **Language:** TypeScript
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com)
- **State & Animation:** [Framer Motion](https://framer.com/motion)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering)

### 3D & Graphics
- **3D Engine:** [Three.js](https://threejs.org)
- **React Integration:** [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber), [@react-three/drei](https://github.com/pmndrs/drei)

### Deployment
- **Hosting:** [Vercel](https://vercel.com)
- **CDN:** Vercel Edge Network
- **Database:** Ready for integration (Supabase, PostgreSQL, etc.)

### Development Tools
- **Package Manager:** npm
- **Linting:** ESLint
- **Formatting:** Prettier (configured via hooks)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/chien-sheng-liu/matarix-data-studio.git
cd studio

# Install dependencies
npm install
```

### Development Server

```bash
# Start dev server on port 3008
npm run dev
```

Open [http://localhost:3008](http://localhost:3008) to view the site.

### Production Build

```bash
# Create optimized production build
npm run build

# Start production server
npm start
```
